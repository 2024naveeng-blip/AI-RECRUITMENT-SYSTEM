# AI Recruitment System

## 📌 Project Description
This project is an AI-based recruitment system designed to automate the hiring process. It analyzes resumes, matches candidates with job roles, and stores data efficiently using a database.

---

## 🚀 Project Flow (Step-by-Step)

1. User uploads or inputs candidate details/resume
2. The system processes the data using AI logic
3. Resume data is analyzed and filtered
4. Candidates are matched with job roles
5. Results are stored in the database
6. Output is displayed to the user

---

## 🧠 Working of the System

- The system reads candidate data
- It uses processing logic to analyze skills and experience
- Based on matching criteria, it selects suitable candidates
- All processed data is saved for future use

---

## 📂 Project Structure (Files Explanation)

### 1. `main.py`
- Main entry point of the project
- Controls the overall flow of the system
- Connects all modules together

### 2. `database.py`
- Handles database operations
- Stores and retrieves candidate data
- Manages tables and records

### 3. `processor.py`
- Contains AI/logic part of the project
- Processes resumes and performs matching
- Filters candidates based on criteria

### 4. `requirements.txt`
- Contains all required Python libraries
- Used to install dependencies

### 5. `system_data.db` & `talent_ai.db`
- SQLite database files
- Store candidate and system data
